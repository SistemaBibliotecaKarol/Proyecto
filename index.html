Sistema de Gestión de Biblioteca

Proyecto desarrollado en Python

Descripción

Este proyecto permite administrar una biblioteca mediante programación orientada a objetos. Incluye el registro de libros, estudiantes y docentes, además del almacenamiento de información en una base de datos SQLite.

Funcionalidades

  • Registro de libros.
  • Registro de estudiantes.
  • Registro de docentes.
  • Lista enlazada para almacenar libros.
  • Ordenamiento por título.
  • Almacenamiento de datos en SQLite.
  • Consulta de libros registrados.

Tecnologías utilizadas

  • Python
  • Programación Orientada a Objetos
  • SQLite
  • GitHub
  • GitHub Pages
